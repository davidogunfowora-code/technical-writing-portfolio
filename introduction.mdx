# Technical Writing Portfolio

I specialize in documenting complex data systems — translating ETL pipelines, data warehouse architectures, reporting frameworks, and campaign analytics environments into clear, structured references for both technical and non-technical audiences.

This portfolio contains eight sample documents built on anonymized data architecture drawn from nonprofit fundraising and digital marketing environments. Each document was produced by interviewing subject matter experts, extracting institutional knowledge, and translating complex technical systems into structured references that analysts, account managers, and client strategy teams can navigate independently.

## Clearfield Data Solutions

A fictional nonprofit fundraising analytics platform. Documents cover the full data lifecycle from client file ingestion through Snowflake warehousing, Alteryx ETL processing, and Power BI reporting.

- **Data Architecture Overview** — End-to-end system map across all five architecture layers
- **Data Mapping Guide** — Schema definitions, PII flagging protocols, and column naming standards
- **Fax Source Automation ETL Process** — Alteryx workflow documentation including input/output specs and deduplication logic
- **Source Code Structure Reference Guide** — Campaign attribution decoding guide with position-by-position field definitions
- **Campaign Performance Report User Guide** — Metric definitions, filter reference, and analyst interpretation guide
- **Matchback Analysis Process & Methodology Guide** — Attribution methodology covering response windows, direct vs indirect gifts, and output interpretation
- **Donor Lifecycle Segment Reference Guide** — All seven segment definitions, progression scenarios, and KPI benchmarks

## Meridian Digital Group

A fictional performance marketing and analytics agency. Documents cover cross-channel digital campaign attribution combining Salesforce, Snowflake, Tableau Prep, and Excel workflows.

- **Attribution Report Process & Methodology Guide** — Complete ten-section guide covering the full six-step ETL workflow and fifteen metric definitions
